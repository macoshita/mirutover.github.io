--- layout: post title: CentOS 6.3でvim最新版をインストール date: 2012-08-29 11:10:17 UTC updated: 2012-08-29 11:10:17 UTC comments: false categories: blog tags: CentOS Vim --- 環境：CentOS6.3

ソースコードをダウンロード、インストールする。
mercurialとpythonが入る。

yum install python-setuptools, python-devel, python, ncurses-devel
easy_install mercurial
hg clone https://vim.googlecode.com/hg/ vim
cd vim/src
./configure --enable-multibyte
make
make install

configureについて、他のブログを見ていると、
--with-features=huge
--enable-xim
--enable-fontset
--disable-selinux
とかしているところもあったけど、ちゃっと調べて自分に必要かどうかすぐ判断できなかったので、一旦外しておいた。


./configure --enable-multibyteが抜けてたり、ncurses-develはデフォルトでは入ってないらしかったり、いろいろ微妙だったので修正しました；
