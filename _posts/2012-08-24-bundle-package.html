bundle package
これを叩くと、vendor/cacheに各種ライブラリのgemがキャッシュされる。
vendor/bundleを含まないようにアーカイブにしてデプロイ先に置いた後、
bundle install --local
と叩けば、rubygems.orgとかにアクセスせず、vendor/cacheのgemを利用してインストールされる。
デプロイ速度向上やgemが無くなった時に役立つのかな。
